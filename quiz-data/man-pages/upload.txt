NAME
    upload - send a local file from operator to target host

SYNOPSIS
    upload [local_path]
    upload [local_path] [remote_path]

DESCRIPTION
    Uploads files to the current Beacon host.
    If remote path is omitted, destination is current working directory on target.

ARGUMENTS
    [local_path]    File path on operator side.
    [remote_path]   Optional destination path on target side.

EXAMPLES
    upload [FilePath]
    upload [FilePath] C:\\ProgramData\\[FileName]
