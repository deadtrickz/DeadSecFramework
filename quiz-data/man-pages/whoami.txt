NAME
    whoami - print effective username for current security context

SYNOPSIS
    whoami

DESCRIPTION
    Displays the account name of the current user/token context.

WINDOWS VARIANTS
    whoami /all      Full token details (groups, privileges, SID).
    whoami /groups   Group membership.
    whoami /priv     Privileges.

EXAMPLES
    whoami
    whoami /all
