NAME
    note - assign operator note/tag text to a Beacon

SYNOPSIS
    note [text]

DESCRIPTION
    Adds or updates a descriptive note for the current Beacon (owner, role, purpose, status, etc.).

ARGUMENTS
    [text]    Free-form note text.

EXAMPLES
    note [Text]
    note [Username] on [Hostname] - local admin
