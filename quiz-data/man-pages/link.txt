NAME
    link - connect this Beacon to another Beacon over SMB/TCP pivoting channel

SYNOPSIS
    link [target]
    link [target] [port]

DESCRIPTION
    Establishes a pivot link to another reachable Beacon. Traffic for the linked Beacon can route
    through the current session.

ARGUMENTS
    [target]    Target host (IP or hostname) where pivot beacon is reachable.
    [port]      Optional listener/transport port when required by the channel.

RELATED
    unlink      Remove the pivot link.

EXAMPLES
    link [IP]
    link [Hostname]
    link [IP] [Port]
