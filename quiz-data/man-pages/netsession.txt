NAME
    netsession - display or manage SMB sessions on a server

SYNOPSIS
    net session
    net session \\[Client]
    net session \\[Client] /delete

DESCRIPTION
    Shows active sessions connected to local server service (LanmanServer).
    Can also terminate a specific client session.

ARGUMENTS
    \\[Client]   Client machine name/IP with active session.
    /delete      Ends the selected session.

EXAMPLES
    net session
    net session \\[IP]
    net session \\[Hostname] /delete
