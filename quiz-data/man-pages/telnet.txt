NAME
    telnet - user interface to the TELNET protocol

SYNOPSIS
    telnet [Host] [Port]

DESCRIPTION
    Commonly used in security labs for simple banner grabs and raw TCP testing.

EXAMPLES
    telnet [IP] [Port]
    telnet 192.168.1.200 22
