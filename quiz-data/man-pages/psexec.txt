NAME
    PsExec - execute processes on remote Windows hosts

SYNOPSIS
    PsExec.exe \\[Target] [options] [command]

DESCRIPTION
    Runs a command remotely over SMB/service control style execution.

COMMON OPTIONS
    -u [User]        Username for remote auth.
    -p [Password]    Password for remote auth.
    -h               Run with elevated token when available.
    -i               Run interactively (session context).
    -s               Run as SYSTEM.
    -d               Do not wait for process to terminate.

EXAMPLES
    PsExec.exe \\[IP] cmd /c whoami
    PsExec.exe \\[Hostname] -u [Domain]\\[User] -p [Password] hostname
    PsExec.exe \\[IP] -s cmd /c ipconfig
