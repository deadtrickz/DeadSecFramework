NAME
    tasklist - list running processes and task details

SYNOPSIS
    tasklist
    tasklist /fi "[Filter]"
    tasklist /s [Target] /u [Domain]\\[User] /p [Password]

DESCRIPTION
    Enumerates running processes locally or remotely with optional filters.

COMMON OPTIONS
    /fi "[Filter]"              Apply a filter (PID, IMAGENAME, STATUS, etc.).
    /svc                         Show services hosted in each process.
    /v                           Verbose details.
    /s [Target]                  Query remote host.

EXAMPLES
    tasklist
    tasklist /fi "IMAGENAME eq lsass.exe"
    tasklist /s [IP] /u [Domain]\\[User] /p [Password]
