NAME
    keylogger - capture keystrokes via Beacon post-ex job

SYNOPSIS
    keylogger
    keylogger [pid] [x86|x64]

DESCRIPTION
    Starts a keystroke logger either in a spawned temporary process or injected into a chosen PID.

ARGUMENTS
    [pid]         Optional process ID for injection mode.
    [x86|x64]     Optional architecture hint when pid is supplied.

RELATED
    jobs          List running jobs.
    jobkill       Stop a running job.

EXAMPLES
    keylogger
    keylogger [PID] x64
