NAME
    systeminfo - display detailed system configuration information

SYNOPSIS
    systeminfo
    systeminfo /s [Target]
    systeminfo /s [Target] /u [Domain]\\[User] /p [Password]

DESCRIPTION
    Prints OS build, patch level, boot time, hardware, domain/workgroup, and hotfix summary.

COMMON OPTIONS
    /s [Target]                  Query remote host.
    /u [Domain]\\[User]           Alternate credentials.
    /p [Password]               Password for /u.
    /fo [TABLE|LIST|CSV]        Output format.

EXAMPLES
    systeminfo
    systeminfo /s [IP]
    systeminfo /s [Hostname] /u [Domain]\\[User] /p [Password]
