# Stories Catalog

All bundled stories live in:
- `quiz-data\stories\*.json`

## Authoring Notes
- Keep answers exact and case-sensitive.
- Add `alternate_answers` for syntax variants you want accepted.
- For custom stories with custom tools, document usage in this file.

## File Path Updates
Legacy documentation and old folders may reference `Stories\...` or `vs-exe-launcher\Stories\...`.
Use `quiz-data\stories\...` as the canonical path.
